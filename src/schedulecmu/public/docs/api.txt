Important API Calls
(All start with "/api")
--

/* GET Course object of course with ID <courseID> */
e.g. /courses/50b64447050c1f770a0004e4

/courses/<courseID>


/* GET the metadata (no class details) of a course <dept>-<num> */
e.g. /courses?number=15-251

/courses?number=<dept>-<num>


/* GET the user object with the auth token <fb_auth_token> */
e.g. /api/users/fbid?auth_token=23092tbjb247oklntg4

/api/users/fbid?auth_token=<fb_auth_token>
